4.if-elif-else结构

if 条件语句1：

语句块2

elif 条件语句2:

语句块2

else:

​ 语句块3

执行过程，先判断条件语句1是否成立，如果成立执行语句块1，然后整个if结构结束。如果不成立，再判断条件语句2是否成立如果成立，执行语句块2，然后整个if结构结束。如果条件语句2不成立，执行语句块3

注意：if-leif-else解构中，elif可以根据情况有多个，else可以省略，如果有else，else写在最后

三、for循环

python中的循环结构有两种：for循环和while循环

当某个操作/某段代码需要重复执行的时候，就需要用到循环结构

1.for循环结构

语法：

for 变量 in 序列：

循环体

说明：

for–>关键字，写法固定

变量–>变量名，按照要求命名

in–>关键字，写法固定

序列->要求是一个值，而且必须是一个序列(容器)，比如：字符串、列表、字典、集合、元祖、迭代器、范围

循环体->和for保持一个缩进的代码段，

功能就是循环体会被重复执行

执行过程:让变量取序列中取值，一个个的取完为止

每取一个值执行一次循环体

x = a

a

x = b

b

x = c

c

四、while循环

1.while循环结构

python中没有do while

语法：

while 条件语句：

​ 循环体

说明：

while ->关键字

条件语句 ->和if后面的条件语句要求一样，可以是任何有值的表达式(不能是赋值语句)

： ->固定的写法

循环体 -> 重复执行的代码段，循环体要和while保持一个缩进

执行过程：判断条件语句的结果是否为True，为True执行循环体，执行完循环体，再判断条件语句是否为True，为True又执行循环体…一次类推，知道判断语句的结果为False，循环结束

注意：while True: —>死循环
