html的基础内容

###一、表单标签和input标签

####1.表单标签: form
这个标签是作为一个容器，来收集和提交这个标签中的其他相关标签的数据。一般不会单独使用。

action属性：设置提交路径(接口)
method属性：设置请求方式(post/get)

####2 input标签
input标签是单标签
input标签可以单独使用，也可以和form配合使用
type属性：这个值不一样，input的功能就不一样

#####2.1 文本输入框(type:text) 和密码输入框(type: password)
type - text(明文输入框)
type - password (密文输入框)

name - 随便命名，用来对输入的内容进行说明。
提交数据的时候以name:value形式进行提交,name值会作为发送请求的参数名，value就参数值
value - 指向的就是文本输入框中的内容
placeholder - 占位符(提示信息)
maxlength - 限制输入框能输入的最大的字符个数

####2.2 单选按钮
type - radio
name - 注意：同一类信息的name需要设置成一样的值，才能做到，选择其中一个让别的取消选中
value - 需要设置值，来告诉别人选中按钮对应值或者意义
checked - 设置默认选中状态

####2.3 复选按钮
type - checkbox
name - 如果是同一类数据，name值应该设置成别人选中按钮对应值或者意义
checked - 设置默认选中状态

####2.4 普通按钮
type - button
value - 设置按钮上显示的文字
disabled - 禁用

####2.5 提交按钮
type - submit
可以自动提交当前form标签中的所有设置了name属性的标签的值

####2.6 重置按钮
type - reset
可以将当前form标签中所有的标签的设置为初始状态的值

###二、下拉菜单和多行文本域
1.下拉菜单: select - option -
name - 提交的时候用来区分不同的数据
value - 提交的时候提交的具体数据
内容 - 显示的部分(只能是文字)
selected - 设置默认选中

对下拉列表进行分组： optgroup
label属性 - 设置分组名

2.多行文本域: textarea
输入的内容可以多行显示
rows - 设置一屏最多能显示的行数
cols - 设置一行的最大的列数
内容 - 相当于value

####三、div和span
div和span标签都是无语义标签，一般用来对网页中的标签进行分组和分块用的
一个div占一行；多个span可以在一行显示
